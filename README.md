

## 一、前言

本小程序用于连接MQTT,可以配置后台配置数据保存信息,自行填写订阅主题和发布信息内容.

## 二、项目简介

本项目采用前后端分离技术

​	前端 微信小程序 + nginx

​	后端 java + Tomcat +数据库

在使用时只需改配置访问后台地址即可.
